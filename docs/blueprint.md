# **App Name**: ProManager

## Core Features:

- Tenant-Specific Workspaces: Allow users to create isolated workspaces, ensuring data privacy and customization for each tenant.
- Project and Task Management: Enable users to create, assign, and track projects and tasks with due dates, priorities, and descriptions.
- Team Collaboration: Allow project owners to invite team members to collaborate on specific projects and tasks, with role-based access control.
- Progress Tracking: Provide visual dashboards and reports to monitor project progress, task completion rates, and team performance.
- Review Summaries: Generate a project review summary, noting tasks completed vs tasks not completed. LLM acts as a tool to choose relevant highlights and action items for the review document.
- Courses Section: Dedicated section to host and display courses related to project management.
- Reviews Section: Dedicated section for user testimonials and reviews of the platform and project courses.

## Style Guidelines:

- Primary color: Professional blue (#3498DB) for a sense of trust and reliability.
- Background color: Light gray (#ECF0F1), providing a clean and modern canvas.
- Accent color: Green (#2ECC71) to highlight key actions and progress indicators.
- Body and headline font: 'Inter' sans-serif, for a modern, neutral, objective appearance that is highly readable.
- Use a consistent set of professional icons to represent different project tasks, resources, and tools.
- A clear, intuitive layout that balances detailed information with high-level overviews for ease of navigation.
- Subtle transitions and animations to enhance user experience without being distracting.
